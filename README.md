# Logistic Algorithms

Implementation of classical optimization and pathfinding algorithms applied to logistics problems, developed as part of university coursework.

## Algorithms

| Algorithm | Description |
|-----------|-------------|
| **A\*** | Heuristic-based shortest path search |
| **Dijkstra** | Single-source shortest path in weighted graphs |
| **Bee Colony** | Swarm intelligence metaheuristic for optimization |
| **Simplex** | Linear programming solver for resource allocation |

## Requirements

- C++ compiler

## Usage

Each algorithm is self-contained in its own folder. Navigate to the desired folder and compile:
```bash
g++ -o output main.cpp
./output
```

## Context

These implementations were used in a university logistics course to demonstrate algorithmic approaches to route optimization, scheduling, and resource planning.
